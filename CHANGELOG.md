# Changelog

## 1.0.4
- Точность GPS для device tracker теперь берется из ответа GWM API, если API возвращает поле точности.
- Если API не возвращает точность или значение некорректное, используется прежний fallback 50 метров.

## 1.0.3
- Исправлена авторизация: добавлены мобильные HTTP-заголовки, чтобы GWM CloudWAF не блокировал запрос логина.
- Исправлена отправка подписанного JSON-тела логина.
- Пароль больше не изменяется пробельной нормализацией.
- Config flow использует стабильный `device_id` на всех шагах настройки.
- Добавлена расширенная диагностика отказов авторизации GWM API.
- Версия устройства в Home Assistant теперь соответствует версии интеграции.

## 1.0.3-beta.2
- Добавлены мобильные HTTP-заголовки для обхода блокировки CloudWAF `418` на запросе логина.

## 1.0.3-beta.1
- Исправлена отправка тела запроса логина: JSON теперь отправляется ровно в том виде, в котором подписывается.
- Пароль больше не обрезается пробельной нормализацией.
- Config flow использует стабильный `device_id` на всех шагах настройки.
- Добавлена расширенная диагностика отказов авторизации GWM API в журнал Home Assistant.

## 1.0
- Первый стабильный релиз интеграции
